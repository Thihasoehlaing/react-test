# React Test Games

## Technology
- Vite
- React,
- TailwindCss
- Tailwind Element

## Project Setup

### Clone repository to your directory
```sh
git clone git@github.com:Thihasoehlaing/react-test.git
```

### Install dependencies
```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
